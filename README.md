# A Void-based Distribution of Zombie Stories GNU/Linux!!
# What is Zombie Stories GNU/Linux?
It's a Linux Distribution (based on Void Linux) for Zombie Stories (Roblox) Players and Linux users who's using Void Linux.
# Features
- Live ISO provided for users who want to use Zombie Stories GNU/Linux without installing
- Users to login:
  Username: anon
  Password: voidlinux
(For XFCE Live ISO only)
- Rootfs Tarball provided for manual installation
- Xinitrc config, Install Desktop Environments, etc scripts (Expected to be added on Beta release)
- Codename for Zombie Stories GNU/Linux is going to be added (Beta release on Feburary 2026 also)
- Other features (coming soon)
